#### 项目目标
- 快速开发
- 灵活配置
- 方便维护
- 快速定位问题
- 可持续集成
- 快速部署
- 集成分布式缓存
- 集成分布式事务
- 集成分布式数据库
- 集成分布式消息队列
- 集成分布式任务调度系统
- 可监控项目运行指标
- 可监控数据库运行指标
- 数据CRUD日志版本记录
- HTTP请求监控
